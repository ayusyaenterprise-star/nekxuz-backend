================================================================================
                            QUICK TROUBLESHOOTING
================================================================================

Images not loading?
  → Check: public_html/assets/cataloges/ exists
  → Verify .htaccess is in public_html/
  → Test: curl https://nekxuz.in/assets/cataloges/honey%20almond%20-600/1.jpg

Routes return 404?
  → .htaccess might be missing or Apache mod_rewrite disabled
  → Contact Hostinger support to enable mod_rewrite
  → Or try uploading .htaccess separately

Slow loading?
  → Check gzip enabled: curl -H "Accept-Encoding: gzip" -v https://nekxuz.in
  → Clear browser cache (Ctrl+Shift+Delete)
  → Check backend at: https://nekxuz-backend-oqcn.onrender.com/api/health
