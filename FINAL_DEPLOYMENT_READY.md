## ❓ If Something Goes Wrong

**Images not showing?**
- Check `/assets/cataloges/` exists in `public_html`
- Verify file permissions are readable (644 for files, 755 for folders)

**Routes not working?**
- Ensure `.htaccess` is in `public_html/` root
- Check Apache `mod_rewrite` is enabled in Hostinger

**Backend not connecting?**
- Verify Render server is running
- Check network requests in browser DevTools

**Checkout failing?**
- Verify Razorpay credentials in backend
- Check Firebase project settings
