## 🆘 If Something Goes Wrong

**Common issues & fixes:**

| Problem | Cause | Fix |
|---------|-------|-----|
| "Cannot GET /" | index.html missing/wrong location | Re-upload to /public_html/ root |
| Styles broken | CSS files failed to load | Check console (F12) for 404 errors |
| Images missing | /assets/ folder not uploaded | Re-upload entire /assets/ folder |
| API fails | Backend not running | Check https://dashboard.render.com |
| Very slow upload | Internet bandwidth limited | Use wired connection or wait |
