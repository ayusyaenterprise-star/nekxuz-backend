## 📋 DEPLOYMENT STEPS (QUICK REFERENCE)

### Render Backend (15 minutes)
```
1. Create PostgreSQL database on Render
2. Push code to GitHub
3. Create Web Service on Render
4. Set environment variables
5. Deploy (5-10 min build time)
6. Get API URL
```

### Hostinger Frontend (20 minutes)
```
1. Update API_BASE_URL in src/App.js
2. Run: npm run build
3. Upload build/ folder to public_html
4. Create .htaccess for React routing
5. Test at yourdomain.com
```
