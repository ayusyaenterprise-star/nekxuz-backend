## 🆘 TROUBLESHOOTING

### Issue: Build keeps failing
**Check**: Go to Deployments → Click failed build → Read logs
**Common causes**: Missing environment variables, database not connected

### Issue: Deployment takes > 15 minutes
**Normal**: First deployment can take longer due to build cache
**Solution**: Be patient, it will complete

### Issue: "Cannot connect to database"
**Problem**: PostgreSQL not set up or connection string wrong
**Solution**: Add DATABASE_URL to environment variables in Render Settings

### Issue: Service keeps restarting
**Problem**: Application crashes due to missing config
**Check**: Application logs for specific error message
