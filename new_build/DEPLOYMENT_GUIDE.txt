📋 TROUBLESHOOTING:
════════════════════════════════════════════════════════════════════════

❌ PROBLEM: Page shows old/cached version
   SOLUTION: 
   • Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
   • Clear browser cache completely
   • Try incognito/private window
   • Wait 5-10 minutes for CDN cache to clear

❌ PROBLEM: "Razorpay is not a constructor" error
   SOLUTION:
   • This is already fixed in this build
   • Make sure you uploaded the static/ folder
   • Check browser console (F12) for errors

❌ PROBLEM: Payment still shows as "test" mode
   SOLUTION:
   • Backend has production keys on Render ✅
   • This build connects to that backend ✅
   • Clear all browser cache and hard refresh
   • Try in a private/incognito window

❌ PROBLEM: Products not loading
   SOLUTION:
   • Check network tab (F12) for API errors
   • Verify backend is running: https://nekxuz-backend.onrender.com
   • Check if you can see orders API response
