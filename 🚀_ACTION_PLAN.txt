TROUBLESHOOTING DURING STEPS:
═════════════════════════════════════════════════════════════════════════════

Problem: npm run build fails
─────────────────────────────────
Solution:
  • Delete node_modules folder
  • Run: npm install
  • Run: npm run build

Problem: Upload to Hostinger is slow
──────────────────────────────────────
Solution:
  • Hostinger might be busy
  • Wait a few minutes
  • Try uploading again
  • Use FTP instead of File Manager

Problem: Website shows old content after upload
──────────────────────────────────────────────
Solution:
  • Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
  • Clear browser cache
  • Wait 5 minutes for CDN to update
  • Try different browser

Problem: Products don't load on website
─────────────────────────────────────────
Solution:
  • Check Network tab (F12) for failed requests
  • Verify backend is running (test URL in address bar)
  • Check console for error messages
  • Ensure API_BASE_URL is correct in code

Problem: Razorpay modal doesn't appear
──────────────────────────────────────
Solution:
  • Check Network tab for failed requests
  • Verify Razorpay script loaded
  • Check console for errors
  • Verify payment keys in backend
