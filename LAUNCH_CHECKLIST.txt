 🚀 DEPLOYMENT STEPS (Choose One Path)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PATH A: Traditional VPS (DigitalOcean Droplet)
────────────────────────────────────────────

[ ] 1. Create Droplet
      $ ssh root@YOUR_IP
      
[ ] 2. System Setup (10 mins)
      $ sudo apt update && sudo apt upgrade -y
      $ sudo apt install -y nodejs npm postgresql nginx certbot python3-certbot-nginx
      
[ ] 3. Clone Repository (5 mins)
      $ git clone YOUR_REPO
      $ cd nekxuz
      $ npm install
      
[ ] 4. Database Setup (10 mins)
      $ sudo -u postgres psql
      $ CREATE DATABASE nekxuz_prod;
      $ CREATE USER nekxuz WITH PASSWORD 'your_password';
      $ npx prisma migrate deploy
      
[ ] 5. Environment Configuration (5 mins)
      $ cp .env.example .env.production
      $ nano .env.production
      (Fill in all required values)
      
[ ] 6. SSL Certificate (5 mins)
      $ sudo certbot certonly --standalone -d nekxuz.com
      
[ ] 7. Build Frontend (10 mins)
      $ npm run build
      
[ ] 8. Start Backend with PM2 (5 mins)
      $ npm install -g pm2
      $ pm2 start server.js --name "nekxuz"
      $ pm2 save && pm2 startup
      
[ ] 9. Configure Nginx (10 mins)
      $ sudo nano /etc/nginx/sites-available/nekxuz
      (Use config from DEPLOYMENT_GUIDE.md)
      $ sudo ln -s /etc/nginx/sites-available/nekxuz /etc/nginx/sites-enabled/
      $ sudo nginx -t
      $ sudo systemctl restart nginx
      
[ ] 10. Verify Deployment (5 mins)
       $ curl https://nekxuz.com/health
       $ curl https://nekxuz.com

TOTAL TIME: ~1 hour


PATH B: Docker with Docker Compose
──────────────────────────────────

[ ] 1. Install Docker & Docker Compose
      $ curl -sSL https://get.docker.com | sh
      $ sudo curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
      
[ ] 2. Prepare Environment
      $ cp .env.example .env.production
      $ nano .env.production
      
[ ] 3. Start Services
      $ docker-compose up -d
      
[ ] 4. Run Migrations
      $ docker-compose exec nekxuz npx prisma migrate deploy
      
[ ] 5. Verify
      $ docker-compose ps
      $ docker-compose logs -f

TOTAL TIME: ~20 minutes

IF BACKEND DOWN:
  $ lsof -i :3002                    # Check if running
  $ tail -f /tmp/backend.log         # View logs
  $ pm2 restart nekxuz               # Restart
  $ docker-compose restart nekxuz    # Or Docker

IF FRONTEND DOWN:
  $ pkill -f "serve -s"              # Kill old process
  $ npm run build                    # Rebuild
  $ serve -s build -l 3004 &         # Restart
  $ docker-compose restart nekxuz    # Or Docker

IF DATABASE DOWN:
  $ pg_isready -h localhost          # Check connection
  $ sudo service postgresql restart  # Restart PostgreSQL
  $ sudo docker-compose logs postgres # Or Docker logs

IF PAYMENT FAILING:
  1. Check Razorpay dashboard
  2. Verify API keys in .env
  3. Check if in live mode
  4. Review error logs
  5. Contact Razorpay support

FULL ROLLBACK:
  $ docker-compose down              # Stop all services
  $ docker system prune              # Clean up
  $ git checkout previous_commit     # Go to previous version
  $ docker-compose up -d              # Restart
